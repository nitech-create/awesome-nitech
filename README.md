# Awesome NITech

名古屋工業大学に関連する便利なツール・サイト・スマホアプリなどをまとめています。

プルリクエストは歓迎です。自分のプロジェクトを掲載されたくない、消してほしいなどの要望があれば Issue または Pull Request で知らせてください。

## 公式サイト・サービス

- [名古屋工業大学](https://www.nitech.ac.jp/) - 名古屋工業大学の公式サイト。ニュース、大学案内、入試、学生生活、研究・産学官連携など。
- [情報基盤センター / サイバーセキュリティセンター](https://www.cc.nitech.ac.jp/) - 学内 IT サービス、障害情報、FAQ、設定手順など。
  - [学生ポータル](https://www.cc.nitech.ac.jp/stn-portal-login.html) - 学生向けポータルサイトのログイン案内。
  - [学生用電子掲示板](https://www.cc.nitech.ac.jp/stn-bbs.html) - 学生向けのお知らせを確認できる電子掲示板。
  - [NITechピロリン](https://www.cc.nitech.ac.jp/service/students/pyrroline.html) - 授業打刻や学生用電子掲示板を利用できる公式アプリ。
  - [Microsoft Azure Dev Tools for Teaching](https://www.cc.nitech.ac.jp/service/students/ms-imagine.html) - 開発者向け Microsoft 製品の提供サービス。
- [シラバス検索](https://syllabus.ict.nitech.ac.jp/) - 授業科目のシラバスを検索できるサービス。
- [名古屋工業大学図書館](https://www.lib.nitech.ac.jp/) - 図書館の公式サイト。蔵書検索、利用案内、電子資料など。
- [名古屋工業大学学術機関リポジトリ](https://nitech.repo.nii.ac.jp/) - 名古屋工業大学の研究・教育成果を公開するリポジトリ。

## Contributing

掲載する項目は、名古屋工業大学の学生・教職員・卒業生・受験生などに役立つものを対象にしています。

- 公式・非公式は問いません。
- 非公式サービスは、提供者やソースコードが確認できる場合はそのリンクも添えてください。
- 終了・長期間未更新のサービスは、必要に応じて `~~サービス名~~` のように取り消し線で残してください。
- 説明文は 1 文程度で、何ができるかを簡潔に書いてください。
